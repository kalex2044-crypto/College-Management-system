# 🎓 College Management System (Frontend Only)

A responsive and modern College Management System built using HTML, CSS, and JavaScript.  
This project demonstrates frontend development skills including UI design, form validation, dynamic calculation, and responsive layout.

---

## 📌 Project Overview

The College Management System allows students to:

- Login securely
- Calculate attendance percentage
- View semester results
- Check fees structure
- Read important notices

This is a frontend-only project (no backend/database).

---

✅ Professional Responsive UI
